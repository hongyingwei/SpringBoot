## 自定义 SpringBoot Starter
- 项目 customize-spring-boot-starter-example 包含两个模块
    - spring-boot-starter-myhello: 
        - 提供了 HelloService 类，及其自动装配的 starter
        - 打包安装到本地仓库
          - mvn clean install
    - hello-app: 
        - 调用 spring-boot-starter-myhello 库及其 starter 的 app
